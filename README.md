# EasyBuy Uganda - Complete E-Commerce System

## Overview
EasyBuy Uganda is a fully functional e-commerce platform built specifically for the Ugandan market with MTN and Airtel Mobile Money integration.

## Features
- ✅ Complete user authentication system
- ✅ Product catalog with categories
- ✅ Shopping cart functionality
- ✅ Secure checkout process
- ✅ MTN Mobile Money & Airtel Money integration
- ✅ Order tracking system
- ✅ Wishlist feature
- ✅ Product reviews and ratings
- ✅ Newsletter subscription
- ✅ Coupon/discount system
- ✅ Responsive design for all devices
- ✅ Admin panel for management
- ✅ Uganda-specific data (districts, UGX currency)

## System Requirements
- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache/Nginx web server
- Composer
- SSL certificate (for production)
